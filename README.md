Reconhecimento de Endometriose

Autor: Victor Gabriel Mendes Sundermann
Orientador Silvio Jamil Ferzoli Guimaraes

Trabalho desenvolvido em parceria entre o laboratorio IMScience e o Hospital da Baleia
Outros Contribuidores:

Objetivos:
- Aplicar metodos de reconhecimento de imagem para segmentar endometriose em imagens de Ressonancia Magnetica
- - Deep Learning
- - - CNN, RNN
- - - Visual Transformers
- - Superpixel
- - Segmentação semantica
- - Morfologia
- Utilizar imagens de RM para gerar um modelo 3D da pelve para melhor visualizar as lesoes

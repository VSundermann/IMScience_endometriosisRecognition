Reconhecimento de Endometriose

Autor: Victor Gabriel Mendes Sündermann
Orientador: Silvio Jamil Ferzoli Guimarães

Trabalho desenvolvido em parceria entre o laboratorio IMScience e o Hospital da Baleia

Objetivos:
- Aplicar metodos de reconhecimento de imagem para segmentar endometriose em imagens de Ressonancia Magnetica
- Deep Learning
- - CNN, RNN
- - Visual Transformers
- Superpixel
- Segmentação semantica
- Morfologia
- Visualização 3D da pelve

Este trabalho foi desenvolvido utilizando as imagens do dataset UTHealth - Endometriosis MRI Dataset, disponivel no link:
https://zenodo.org/records/15750762
